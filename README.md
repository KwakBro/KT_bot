KaKaotalk Bot 
=============
본 코드는 Violet XF 님의 Messenger bot에서 동작하는 코드. 

목적
====
주식 거래시 MTS와 카카오톡을 왔다갔다 하며 정보를 얻는것이 불편한 것에서 착안  
자동응답 기능을 이용하여 카카오톡 에서도 실시간으로 정보를 얻어오기 위함.
